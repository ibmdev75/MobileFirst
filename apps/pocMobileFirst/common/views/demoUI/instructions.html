Instructions pour ajouter des cas d'évaluations

Cette application est développée sous AngularJS et Mobile Angular UI
Un exemple de page existe dans l'application sous le nom de TEMPLATE :

views/pocMF/template.html
js/controllers/template.js

5 actions pour ajouter un cas d'usage :

  • 1. ajouter une page web dans le dossier views/pocMF
  • 2. ajouter un controlleur Angular, dans le dossier js/controllers
  • 3. ajouter l'option de menu dans le fichier views/sidebar.html
  • 4. ajouter la route vers la nouvelle page et le nouveau controlleur dans le fichier js/app.js
  • 5. ajouter le scripts du controlleur dans le fichier index.html

Liste des autres frameworks :

  • Jquery : manipulation des objets DOM
  • Font Awesome : bibliothèque d'icônes
  • PNotify : affichage de message
  • Angular Loading Bar : barre de progression Http
